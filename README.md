# 💫 About Me:
I’m Sireen, a developer bridging full-stack engineering with applied machine learning. I build high-utility web applications using Next.js, React, Node.js, FastAPI, and Supabase. On the AI front, my work ranges from deploying CNNs and YOLO models with PyTorch, TensorFlow, and CUDA, to grounding my foundational knowledge by building neural networks from scratch using NumPy and integrating LLMs via the Groq API. Beyond software, I shoot and edit documentary-style, 4K cinematic video, bringing a strong visual aesthetic to everything I produce
